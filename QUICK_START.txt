📋 STEP-BY-STEP CHECKLIST:

□ 1. Go to https://supabase.com and create account
□ 2. Create new project (takes 2-3 minutes)
□ 3. Go to SQL Editor → New Query
□ 4. Copy & paste ALL of supabase-schema.sql
□ 5. Click "Run" button
□ 6. Go to Project Settings → API
□ 7. Copy your Project URL
□ 8. Copy your anon public key
□ 9. Create .env.local file in project root
□ 10. Add these two lines:

NEXT_PUBLIC_SUPABASE_URL=your_project_url_here
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key_here

□ 11. Restart dev server: npm run dev
□ 12. Test by connecting wallet and waving at someone!
